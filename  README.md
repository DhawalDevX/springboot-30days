# Book & User Management API

A REST API built with Spring Boot

## Tech Stack
- Java + Spring Boot
- MySQL + JPA/Hibernate
- ModelMapper
- Swagger UI
- Validation

## Features
- Book CRUD with Pagination & Sorting
- User Management API
- Global Exception Handling
- DTO Pattern
- Swagger Documentation

## API Endpoints
### Books
- GET /books
- POST /books
- PUT /books/{id}
- DELETE /books/{id}

### Users
- GET /users
- POST /users
- PUT /users/{id}
- DELETE /users/{id}
